# 项目概述 java1011

本项目是基于Springboot和Vue的音乐推荐管理系统，采用了协同过滤算法，为用户提供个性化的音乐推荐服务。系统包含用户和管理员两个角色，分别使用不同的前端界面进行操作。

## 技术栈

### 后端

- Springboot
- MyBatis

### 前端

- Vue
- ElementUI
- ECharts

## 功能模块

### 用户功能

- 首页推荐歌单
- 歌单分类分页界面
- 歌手分类分页界面
- 我的音乐：查看、收藏歌曲
- 音乐搜索：支持按歌手、歌曲、歌单名搜索
- 用户设置：头像修改、用户信息修改
- 歌曲播放：支持进度条拉伸、歌词加载、收藏、下载

### 管理员功能

- 系统首页：展示统计数据
- 用户管理
- 歌手管理
- 歌曲管理：修改音源、歌词、评论
- 音乐上传

## 系统角色

- 用户
- 管理员

## 运行环境
